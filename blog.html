Question-2: What is the difference between innerHTML, innerText, and textContent ?

There are some key defference between innerText , innerHTML and textContent. Question-3: What is event delegation in the DOM?

Event delegation is a technique in JavaScript where instead of attaching event listeners to multiple child elements, you add a single event listener to a parent element. This works because events bubble up through the DOM, allowing the parent to detect events from its children.

Question-4: What is event bubbling in the DOM?

If you set an event listener on multiple nested elements, the event starts bubbling from the deepest element where the event occurred and moves up through its ancestors. This process is called event bubbling. For example, if you have an HTML structure like: parent element > inner parent > child > inner child, and you click on the parent, the event will first check the innermost element with an event listener before propagating upward to the parent elements
